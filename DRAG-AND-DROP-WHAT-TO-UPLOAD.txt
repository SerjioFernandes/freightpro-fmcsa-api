EXACT UPLOAD PROCESS
================================================================

STEP 1: Build Frontend (REQUIRED!)

Open PowerShell:
cd C:\Users\HAYK\Desktop\FreightPro
cd frontend
npm install
$env:VITE_API_URL="https://api.yourdomain.com/api"; npm run build

Wait until it says "Build complete"

---

STEP 2: Open Hostinger File Manager

In Hostinger:
1. Go to "Advanced" section
2. Click "File Manager"
3. Navigate to: domains/yourdomain.com/public_html/

---

STEP 3: Upload Backend

1. Create folder called "api"
2. Open that "api" folder
3. Drag these from your computer:
   - backend/dist/ folder
   - backend/package.json
   - backend/package-lock.json
   - Others/ecosystem.config.js

---

STEP 4: Upload Frontend

1. Go back to public_html/ (root)
2. Delete default index.html if exists
3. Drag ALL files from frontend/dist/:
   - index.html
   - assets/
   - manifest.json
   - (other files)
4. Create .htaccess file with content above

---

STEP 5: Configure Node.js App

1. Go back to Hostinger main menu
2. Advanced ‚Üí Node.js Apps
3. Create new app
4. Fill in settings (see guides)
5. Add environment variables

---

STEP 6: Enable SSL

1. Advanced ‚Üí SSL
2. Enable for www.yourdomain.com
3. Enable for api.yourdomain.com

---

STEP 7: Test

Visit: https://www.yourdomain.com
