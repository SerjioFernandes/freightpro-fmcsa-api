================================================================

Copy these to create your .env files.

================================================================

================================================================
GENERATING SECRETS
================================================================

JWT Secret:
-----------
node -e "console.log(require('crypto').randomBytes(48).toString('hex'))"

Creates 96-character secure random string.

MongoDB Connection:
-------------------
From MongoDB Atlas → Cluster → Connect → Connect your application
Copy connection string and replace username/password.

SMTP Credentials:
-----------------
Use the SMTP username/password provided by your email host (e.g. Hostinger, SendGrid).
For SSL (port 465) set EMAIL_SECURE=true. For STARTTLS (port 587) set EMAIL_SECURE=false.

================================================================
IMPORTANT
================================================================

❌ NEVER commit .env files (in .gitignore)
✅ In production, set env vars in Railway dashboard
✅ Rotate secrets regularly
✅ Use different secrets for dev/staging/production
