================================================================
WHAT IS NODE-RED?
================================================================

Node-RED is a visual programming tool for automation.
It's NOT the same as Node.js!

Node-RED = Visual automation tool (like Zapier)
Node.js = JavaScript runtime for building apps

Your FreightPro backend = Node.js application

================================================================
CAN YOU USE NODE-RED FOR YOUR BACKEND?
================================================================

❌ NO - Node-RED is different from Node.js

Your app needs:
- Express server
- Socket.io WebSocket
- MongoDB connection
- JWT authentication
- File uploads
- Email sending

Node-RED is for:
- Automation workflows
- IoT devices
- Data pipelines
- Visual programming

Your app = Custom Node.js + Express + TypeScript
Can't run in Node-RED directly
