# üßπ How to Clear Browser Cache

## Quick Method (All Browsers):

### Press these keys at the same time:
- **Windows/Linux**: `Ctrl + Shift + Delete`
- **Mac**: `Cmd + Shift + Delete`

This opens the "Clear browsing data" window!

---

## Step-by-Step Instructions:

### üåê Google Chrome:

**Method 1: Keyboard Shortcut**
1. Press `Ctrl + Shift + Delete` (Windows) or `Cmd + Shift + Delete` (Mac)
2. Select **"Cached images and files"**
3. Time range: **"All time"** or **"Last hour"**
4. Click **"Clear data"**

**Method 2: Menu**
1. Click the **three dots** (‚ãÆ) in top-right corner
2. Go to **"More tools"** ‚Üí **"Clear browsing data"**
3. Select **"Cached images and files"**
4. Time range: **"All time"**
5. Click **"Clear data"**

**Method 3: Hard Refresh (Quick Fix)**
- Press `Ctrl + F5` (Windows) or `Cmd + Shift + R` (Mac)
- This forces browser to reload without cache

---

### ü¶ä Microsoft Edge:

**Method 1: Keyboard Shortcut**
1. Press `Ctrl + Shift + Delete`
2. Select **"Cached images and files"**
3. Time range: **"All time"**
4. Click **"Clear now"**

**Method 2: Menu**
1. Click the **three dots** (‚ãØ) in top-right corner
2. Go to **"Settings"** ‚Üí **"Privacy, search, and services"**
3. Click **"Choose what to clear"**
4. Select **"Cached images and files"**
5. Click **"Clear now"**

**Method 3: Hard Refresh**
- Press `Ctrl + F5` or `Ctrl + Shift + R`

---

### ü¶ä Firefox:

**Method 1: Keyboard Shortcut**
1. Press `Ctrl + Shift + Delete`
2. Select **"Cache"**
3. Time range: **"Everything"**
4. Click **"Clear Now"**

**Method 2: Menu**
1. Click the **three lines** (‚ò∞) in top-right corner
2. Go to **"Settings"** ‚Üí **"Privacy & Security"**
3. Scroll to **"Cookies and Site Data"**
4. Click **"Clear Data"**
5. Select **"Cached Web Content"**
6. Click **"Clear"**

---

### üçé Safari (Mac):

**Method 1: Menu**
1. Click **"Safari"** in menu bar
2. Go to **"Preferences"** ‚Üí **"Advanced"**
3. Check **"Show Develop menu in menu bar"**
4. Click **"Develop"** ‚Üí **"Empty Caches"**

**Method 2: Keyboard**
- Press `Cmd + Option + E` to empty cache

---

## ‚ö° Quick Hard Refresh (No Menu Needed):

If you just want to reload the current page without cache:

- **Windows**: Press `Ctrl + F5` or `Ctrl + Shift + R`
- **Mac**: Press `Cmd + Shift + R`

This forces the browser to reload the page and ignore cache!

## üÜò If Still Not Working:

**Try these additional steps:**
1. Clear cache (as above)
2. **Close browser completely** (all windows)
3. **Reopen browser**
4. **Hard refresh**: `Ctrl + F5`
5. Visit your website again

**Or use Incognito/Private Mode:**
- Chrome: `Ctrl + Shift + N`
- Edge: `Ctrl + Shift + P`
- Firefox: `Ctrl + Shift + P`

Private mode doesn't use cache, so you'll always see the latest version!
